Introduction

The Smart Air Quality Monitoring and Alert System is designed to help people easily track the cleanliness and safety of the air in their surroundings. It uses sensor units placed in different areas to measure air pollution, temperature, and humidity. The collected data is sent to a main device, which uploads it to an online platform for real-time viewing. Whenever the air becomes unsafe .for instance, when pollution or temperature levels rise above normal , the system instantly sends an alert to warn users. This allows quick action to protect health and improve living conditions. By providing accurate and timely information, this project encourages environmental awareness and supports healthier communities in homes, schools, and workplaces.

Project Abstract

This project introduces a Smart Air Quality Monitoring and Alert System that tracks air pollution, temperature, and humidity to ensure a healthy environment. The system collects data through sensor units and sends it to an online platform for real-time viewing. When unsafe air conditions are detected, it immediately alerts users to take quick action. By combining technology with environmental awareness, this project provides a simple and effective way to monitor air quality in homes, schools, and workplaces.

Working Principle

.Working principle descrption:

The Smart Air Quality Monitoring and Alert System operates through a network of sensor nodes, a central control unit, and an online monitoring platform. Each sensor node continuously measures key environmental parameters such as carbon dioxide (CO₂), particulate matter (PM2.5), temperature, and humidity. These nodes use compact microcontrollers to process the data locally before transmitting it to the main control unit via UART serial communication. The central control unit, powered by an ESP32, receives the data from all sensor nodes, analyzes it, and connects to the internet using built-in Wi-Fi. It then uploads the readings to the Adafruit IO cloud dashboard, where users can view real-time trends and insights through dynamic charts and indicators. Whenever the system detects unsafe air conditions or abnormal environmental changes, it instantly triggers visual or audible alerts and sends notifications through the dashboard. This ensures users are informed immediately and can take action to protect their health and surroundings. Overall, the system provides a smart, connected, and reliable solution for monitoring air quality in homes, schools, and workplaces while promoting awareness and environmental safety through modern IoT technology.

    4. Gateway firm ware:

    The following are complete code which enable esp32 to receive data from each node and send it to dash board

     #include 
#include 
#include "Adafruit_MQTT.h"
#include "Adafruit_MQTT_Client.h"

#define WIFI_SSID "xxxxxx"
#define WIFI_PASS "xxxxx"

#define AIO_SERVER      "io.adafruit.com"
#define AIO_SERVERPORT  1883
#define AIO_USERNAME    "xxxxx"
#define AIO_KEY         "xxxxxxx"

// Define Serial2 pins
#define RXD2 16
#define TXD2 17
HardwareSerial Serial2(2);  // UART2

// Create WiFi client and MQTT client
WiFiClient client;
Adafruit_MQTT_Client mqtt(&client, AIO_SERVER, AIO_SERVERPORT, AIO_USERNAME, AIO_KEY);

// Feeds
Adafruit_MQTT_Publish co2_feed     = Adafruit_MQTT_Publish(&mqtt, AIO_USERNAME "/feeds/co2");
Adafruit_MQTT_Publish pm_feed      = Adafruit_MQTT_Publish(&mqtt, AIO_USERNAME "/feeds/pm25");
Adafruit_MQTT_Publish temp_feed    = Adafruit_MQTT_Publish(&mqtt, AIO_USERNAME "/feeds/temp");
Adafruit_MQTT_Publish humidity_feed= Adafruit_MQTT_Publish(&mqtt, AIO_USERNAME "/feeds/humidity");

void setup() {
  Serial.begin(9600);
  Serial2.begin(9600, SERIAL_8N1, RXD2, TXD2);

  delay(2000);
  Serial.println("Connecting to WiFi...");

  WiFi.begin(WIFI_SSID, WIFI_PASS);
  while (WiFi.status() != WL_CONNECTED) {
    delay(500);
    Serial.print(".");
  }
  Serial.println("\nWiFi Connected!");
}

void loop() {
  MQTT_connect();

  if (Serial2.available()) {
    String line = Serial2.readStringUntil('\n');
    line.trim();

    if (line.length() > 0) {
      Serial.print("Received: ");
      Serial.println(line);

      float co2 = getValue(line, "CO2=");
      float pm = getValue(line, "PM=");
      float temp = getValue(line, "TEMP=");
      float hum = getValue(line, "HUM=");

      co2_feed.publish(co2);
      pm_feed.publish(pm);
      temp_feed.publish(temp);
      humidity_feed.publish(hum);
    }
  }
}

// ---------- Helper Functions ----------
float getValue(String data, String key) {
  int index = data.indexOf(key);
  if (index == -1) return 0;
  int end = data.indexOf(';', index);
  String val = (end == -1) ? data.substring(index + key.length()) : data.substring(index + key.length(), end);
  return val.toFloat();
}

void MQTT_connect() {
  int8_t ret;
  if (mqtt.connected()) return;

  Serial.print("Connecting to Adafruit IO... ");
  while ((ret = mqtt.connect()) != 0) {
    Serial.println(mqtt.connectErrorString(ret));
    Serial.println("Retrying MQTT connection in 5 seconds...");
    mqtt.disconnect();
    delay(5000);
  }
  Serial.println("MQTT Connected!");
}
